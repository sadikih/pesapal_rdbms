# Pesapal Junior Dev Challenge 2026 - RDBMS Project

## Description
This project implements a simple Relational Database Management System (RDBMS) in PHP, supporting:

- Table creation
- CRUD operations (Create, Read, Update, Delete)
- Primary and Unique key enforcement
- Basic indexing
- SQL-like interface with interactive methods
- Demonstration via a trivial web application

## Technologies Used
- PHP 8+
- XAMPP (Apache)
- VS Code

## How to Run
1. Clone the repository.
2. Copy the project folder to your XAMPP `htdocs` directory.
3. Start Apache via XAMPP.
4. Open `http://localhost/pesapal_rdbms/index.php` in your browser.
5. Use the web interface to test CRUD operations.
